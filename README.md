# Lionheart Digital - Coming Soon

A coming soon landing page for Lionheart Digital, a Service-Disabled Veteran-Owned Small Business (SDVOSB) providing digital services.

## Overview

This is a simple, static landing page built with HTML and CSS. It features:

- Responsive design for mobile, tablet, and desktop
- Hero section with company branding and description
- Values section highlighting company principles
- SDVOSB certification display
- Accessible markup with proper ARIA labels

## Project Structure

```
├── index.html       # Main HTML page
├── styles.css       # Stylesheet
└── LionheartLogo.png # Company logo
```

## Setup

No build process required. Simply open `index.html` in a browser or serve the directory with any static file server.

```bash
# Example using Python
python -m http.server 8000

# Example using Node.js (npx)
npx serve .
```

## Customization

Before deploying, update the following placeholder:

- **UEI Number**: Replace `[YOUR-UEI-HERE]` in `index.html` with your actual UEI
- **SDVOSB Logo**: Replace the placeholder in the certification section with the official logo
